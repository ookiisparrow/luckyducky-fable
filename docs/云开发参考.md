## 1. tcb CLI 踩坑（本项目已验证·官方不明写）

> 用 CloudBase CLI（`tcb`）从本机命令行部署云函数 / 读写云库，不必每次开微信开发者工具 GUI。配置在项目根 `cloudbaserc.json`（`tcb fn deploy` 自动读 envId / functionRoot / functions）。常用：`tcb env list`（验登录态）/ `tcb fn deploy --all`（批量·首次即建·自动装依赖）/ `tcb fn deploy <name> --force`（覆盖单个）/ `tcb fn invoke <name>` / `tcb db nosql execute --command '<JSON>'`。

- **`tcb fn invoke` 没有 openid 上下文**：依赖 `cloud.getWXContext().OPENID` 的函数（如 login）CLI 调不出用户态，需小程序端调；不依赖 openid 的（getProducts / seedProducts）可 CLI 直接调验数据。带 openid 的端到端验证走 miniprogram-automator 在工具里 `evaluate` 调 `wx.cloud.callFunction`（样板 `scripts/p3-verify.cjs`）。
- **`--force` 更新不会自动在云端装依赖**（只有新建函数才装）→ 已在 `cloudbaserc.json` 每个函数配 `installDependency: true`；invoke 报 `Cannot find module 'wx-server-sdk'` 即此坑（2026-06-10 踩过）。
- **`tcb fn deploy A B` 只部署第一个**（2026-06-11 踩过：`deploy trackEvent getMyProgress` 仅 trackEvent 成功，getMyProgress invoke 报「未找到 Function」）→ 多函数**逐个 deploy** 或 `--all`；部署后逐个 `fn invoke` 验存在。
- **`tcb login` 过期重登**：浏览器授权页**选「微信公众号」扫码**（小程序管理员微信），**不要用腾讯云账号密码**——微信云开发环境挂在微信侧账号体系下（2026-06-10 验证）。
- **`tcb db nosql execute` 的 --command 结构**：`[{"TableName":…, "CommandType":"QUERY|INSERT|UPDATE|DELETE|COMMAND", "Command":"<字符串化的 Mongo 命令>"}]`（以 `tcb db nosql execute -h` 示例为准），不是裸 Mongo 语句（2026-06-11 踩过）。
- 直接改 live 库（`db nosql execute` 写）属敏感操作，按环境权限策略可能需显式授权（本仓另有 `guard-deploy` 部署闸）。

## 2. 数据库 `_openid` 管理端坑（见调试日志）

- 只有**小程序端** `add` 才自动写 `_openid`；**云函数 / 控制台属管理端，`add` 不写 `_openid`**——云函数写业务数据后还要按 `_openid` 查 / 鉴权的，**必须显式写入**（取 `cloud.getWXContext().OPENID`）。
- 不要假设所有数据都有 `_openid`；按 `_openid` 查不回多半是这条。

## 3. 已踩平台限制与解法（功能未受损）

| 限制 | 现象 | 解法 |
| --- | --- | --- |
| HTTP 服务请求体 ~100KB | 控制台传图 UPLOAD_FAIL（调试日志 F） | 图片 ~80KB 分片 |
| 大文件不走普通接口 | 视频传不动 | manager-node 签凭证浏览器直传（失败回落分片） |
| 看板单表一次统计 ≤1000 条 | 千级够用·上万少算 | 上量改预聚合（工程改造·功能不变·见 `待办与债` 容量项） |
| 单文档 512KB | 课程/商品档案远未触及 | — |
| 冷启动 | 久未调用函数首响慢 1–2s | 体验毛刺非故障 |

## 4. 架构形态（一句话 + 指针）

Serverless（无自有实例）：云函数（按调用计费·无常驻）+ 云数据库 + 云存储 + HTTP 访问服务，腾讯云上海地域（ap-shanghai·多副本容灾），云存储自带 CDN（`tcb.qcloud.la` 走边缘节点·上量后大头是视频 CDN 流量·VOD 评估见主规格 P5）。文件目录约定 `products/`（商品图）/`videos/`（课程视频）/`avatars/`（头像）。

**两个「后台」分清**：① 腾讯云开发控制台（机房仪表盘·原始数据/文件/函数日志/账单）＝微信开发者工具「云开发」按钮 或 tcb.cloud.tencent.com（**选「微信公众号」扫码**）；② 自建管理控制台（日常上新/橱窗/看板/发货）＝云开发静态托管，URL 见 `现状与路线` §运维事实（单源），本机开发 `cd packages/admin && npm run dev`。

> 环境 ID / 地域 / 套餐档位 / 到期续费 / 集合数 等客观事实**不在此抄**——单源在 `现状与路线` §运维事实（防与订正后的真值漂移：曾把「2026-12-07/个人版」写错、已在单源订正为「2026-07-11/标准版」）。
