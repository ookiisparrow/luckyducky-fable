# 归档索引（退役层·唤起入口）

> **这是什么**：活文档的「退役层」。已完成过程的边角料（批次账/已关状态/历史规格/审核报告…）从活文档退役到此——**意图与理由留活文档，过程产物退到这里可查**。
> **怎么唤起**：在本表按「关键词/源」找到归档文件 → 读它 / 需要时把片段引回活文档。反向：每份归档标「源活文档」，知道它当年从哪退役。
> **机器守**：`archive-index-synced`（`scripts/check-structure.mjs`）焊死——每份 `archive/*.md` 必在本表登记（防退役了查不到）+ 活文档引用的 `archive/*` 必存在（防悬空指针）。新增归档**必加一行**，否则 `npm run check` 红。
