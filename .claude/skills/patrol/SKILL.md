# 自主巡逻（一轮）

> 本 skill = 元模式 **health 环的自动巡更**（`docs/元模式.md` A5）——/anti-entropy-loop 的 /loop 降档版：单镜头、轻量、每轮独立收口。**只定义一轮**；循环/间隔/唤醒归 /loop。composes `/doc-audit`（镜头 A 真值法）+ `/systematic-debugging`（修 bug）+ `/refactor-batch`（落地），不复述。

## 一轮六步
1. **停轮闸**：`git status --short` 非空 → 停轮上报（工作区有未提交的活，巡逻不卷别人改动），本轮终止。
2. **基线**：`npm run report -- --fast`（秒级）。红且机器可修 → 本轮镜头强制切「修基线」；红因待人（真机/控制台）→ 记账+停轮上报；绿 → 继续。HEAD 干净时不跑全量 check（typecheck/lint 上次 pre-commit 已验；全量留给有改动要提交时，pre-commit 反正会拦）。
3. **定镜头**：读 `docs/巡逻日志.md` 最新一条的镜头字段，按 A→B→C→D→A 轮换；日志缺失/空从 A 起。
4. **亲扫亲核**：按镜头扫（见下），每条 finding 带「文件:行号 + 原文摘录 + 真值来源 + 建议改法」；**声称一律亲核**（对 git/代码/glob 验过才算数）。单轮亲扫，不 fan-out 子代理。
5. **三分诊 → 修**：
   - **机器可修**（文本对齐真相/静态可证/可加守卫）→ 一轮**一个修复批**、小改为限（约 ≤30 分钟）；大 finding 拆轮或挂 `docs/待办与债.md`。修走 `/refactor-batch` 纪律：能上守卫先红后绿，纯文本修正在轮报记明为何无守卫面；`npm run check` 全绿才提交。超时不绿 → 还原、降级挂账、收轮上报。
   - **要人验**（真机/多端/支付/云控制台/需求口径，根因#8）→ 按格式记账（bug → `docs/调试日志.md` 五段格式；债 → `docs/待办与债.md`），不修、不宣称验证过。
   - **拿不准**（动钱/不可逆/口径未定/疑似业务预留）→ 停轮问用户。
   - **镜头外发现** → 记账不修（除非 P0），下轮或专批处理。
6. **轮报 → `docs/巡逻日志.md`**：置顶追加一条（轮次/日期/镜头/基线态/findings 三分诊去向/「与本轮报同批提交」或「跑了·验了·干净」）；随修复批同批提交，无修复批则单独 `chore：巡逻轮报 #N`。零发现 = 负结果照报，不凑数。

## 镜头（轮换 A→B→C→D）
- **A 状态文档准确性**：`git log -20 --oneline` vs `docs/现状与路线.md`/`docs/调试日志.md`/`docs/待办与债.md` 全文 + `docs/重构日志.md` **顶部 3 条**（省 token·全史看 git）——核「声称 vs commit 真相」（`/doc-audit` 真值法切片），揪进度报低/已修未勾/已落未回写。
- **B 台账核对**：`待办与债`/`调试日志` 挂账项逐条问「被后续 commit 悄悄解决没有」「解冻条件达成没有」；达成的勾销/推进，未达成的不催。
- **C 守卫抽检**：从近 5 批提交挑 1 条新守卫做反向自检（篡改→必红→还原→绿）；不红 = 摆设守卫，即重大 finding 走修复批。
- **D 代码红旗有界扫**：CLAUDE §7 red flags 在 `rewrite/` 单包轮转（mp→cloud→admin→agent→site→shared），只报静态可证的。

## 节奏（给 /loop 自定节奏的提示）
干净 2 轮 → 建议拉长间隔（≥30 分钟或睡到下个自然节点）；有修复/基线红 → 正常或缩短。单轮预算 ~15 分钟，超了收敛范围下轮续，不熬长轮。

## 红线
- **不部署、不动云端**（数据库/控制台/工作流）——共用真实环境，巡逻只在仓内。
- 真机/多端/支付层机器证不出（根因#8）→ 记账不修，**绝不宣称「验证过」**。
- 用户验收窗口期（M 批待真机走查/演练/审稿）rewrite 产物**行为面** → 记账不修，不搅用户即将走查的表面。
- 需求变更/疑似业务预留 → 只记账（CLAUDE §7 业务预留分流），不动手。
- 一轮一个修复批、小改为限；修必走 /refactor-batch 全套，被 check 拦 = 修代码不绕闸。
- 逐轮流水账只进 `巡逻日志`；真 finding 才落 `调试日志`/`待办与债`。
- 工作区脏 / 基线「待人」红 / 拿不准 → 停轮问，不硬跑。

> 一句话：一轮 = 闸 → 基线 → 一个镜头亲扫亲核 → 三分诊（一批小修/记账/停问）→ 轮报落册；干净也是交付，循环交给 /loop。
